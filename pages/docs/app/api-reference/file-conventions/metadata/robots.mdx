# robots.txt

Tambahkan atau buat berkas `robots.txt` yang sesuai dengan [Standard Penolakan Robot](https://en.wikipedia.org/wiki/Robots.txt#Standard) di **root** direktori `app` untuk memberi tahu pengindeks mesin pencari URL mana yang dapat diakses di situs Anda.

## Berkas `robots.txt` Statis

```txt filename="app/robots.txt"
User-Agent: *
Allow: /
Disallow: /private/

Sitemap: https://acme.com/sitemap.xml
```

## Hasilkan Berkas Robot

Tambahkan berkas `robots.js` atau `robots.ts` yang mengembalikan objek [`Robots`](#objek-robots).

```ts filename="app/robots.ts" switcher
import { MetadataRoute } from 'next';

export default function robots(): MetadataRoute.Robots {
  return {
    rules: {
      userAgent: '*',
      allow: '/',
      disallow: '/private/',
    },
    sitemap: 'https://acme.com/sitemap.xml',
  };
}
```

```js filename="app/robots.js" switcher
export default function robots() {
  return {
    rules: {
      userAgent: '*',
      allow: '/',
      disallow: '/private/',
    },
    sitemap: 'https://acme.com/sitemap.xml',
  };
}
```

Keluaran:

```txt
User-Agent: *
Allow: /
Disallow: /private/

Sitemap: https://acme.com/sitemap.xml
```

### Objek Robots

```tsx
type Robots = {
  rules:
    | {
        userAgent?: string | string[];
        allow?: string | string[];
        disallow?: string | string[];
        crawlDelay?: number;
      }
    | Array<{
        userAgent: string | string[];
        allow?: string | string[];
        disallow?: string | string[];
        crawlDelay?: number;
      }>;
  sitemap?: string | string[];
  host?: string;
};
```
