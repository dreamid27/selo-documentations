## API yang Tidak Didukung

Edge Runtime memiliki beberapa pembatasan termasuk:

- API Native Node.js **tidak didukung**. Misalnya, Anda tidak dapat membaca atau menulis ke sistem file.
- `node_modules` _dapat_ digunakan, selama mereka mengimplementasikan Modul ES dan tidak menggunakan API Native Node.js.
- Memanggil `require` secara langsung **tidak diizinkan**. Gunakan Modul ES (ES Modules) sebagai pengganti.

Berikut adalah fitur bahasa JavaScript yang dinonaktifkan dan **tidak akan berfungsi:**

| API                                                                                                                             | Deskripsi                                                          |
| ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| [`eval`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/eval)                                       | Mengevaluasi kode JavaScript yang direpresentasikan sebagai string |
| [`new Function(evalString)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Function)               | Membuat fungsi baru dengan kode yang diberikan sebagai argumen     |
| [`WebAssembly.compile`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/compile)         | Kompilasi modul WebAssembly dari kode JavaScript                   |
| [`WebAssembly.instantiate`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/instantiate) | Membuat objek WebAssembly dari modul dan objek impor               |

Dalam kasus yang jarang terjadi, kode Anda bisa mengandung (atau mengimpor) beberapa pernyataan evaluasi kode dinamis yang _tidak dapat dicapai saat runtime_ dan yang tidak dapat dihapus oleh treeshaking.
Anda dapat mengendurkan pemeriksaan untuk mengizinkan file tertentu dengan konfigurasi diekspor pada Middleware atau Rute API Edge Anda:

```javascript
export const config = {
  runtime: "edge", // hanya untuk Rute API Edge
  unstable_allowDynamic: [
    // mengizinkan satu file
    "/lib/utilities.js",
    // gunakan glob untuk mengizinkan semua di dalam modul pihak ketiga function-bind
    "/node_modules/function-bind/**",
  ],
};
```

`unstable_allowDynamic` adalah sebuah [glob](https://github.com/micromatch/micromatch#matching-features), atau sebuah array dari glob, yang mengabaikan evaluasi kode dinamis untuk file-file tertentu. Glob-glob tersebut relatif terhadap folder root aplikasi Anda.

Perlu diingat bahwa jika pernyataan-pernyataan ini dieksekusi pada Edge, _mereka akan menghasilkan error saat runtime_.
