# Styling

Next.js mendukung berbagai cara untuk styling aplikasi Anda, termasuk:

- **CSS Global**: Mudah digunakan dan familiar bagi mereka yang berpengalaman dengan CSS tradisional, tetapi dapat menghasilkan bundel CSS yang lebih besar dan kesulitan dalam mengelola gaya saat aplikasi berkembang.
- **Modul CSS**: Buat kelas CSS yang berlokasi lokal untuk menghindari konflik penamaan dan meningkatkan pemeliharaan.
- **Tailwind CSS**: Kerangka kerja CSS yang berorientasi pada utilitas yang memungkinkan desain kustom yang cepat dengan menggabungkan kelas utilitas.
- **Sass**: Preprosesor CSS populer yang memperluas CSS dengan fitur seperti variabel, aturan bersarang, dan mixin.
- **CSS-in-JS**: Sematkan CSS langsung dalam komponen JavaScript Anda, memungkinkan penggayaan dinamis dan berlokasi.
