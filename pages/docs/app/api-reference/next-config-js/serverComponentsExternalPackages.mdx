# serverComponentsExternalPackages

Ketergantungan yang digunakan dalam [Server Components](/docs/app/building-your-application/rendering/server-components) dan [Route Handlers](/docs/app/building-your-application/routing/route-handlers) akan secara otomatis dibundel oleh Next.js.

Jika suatu ketergantungan menggunakan fitur-fitur khusus Node.js, Anda dapat memilih untuk menyisihkan ketergantungan spesifik dari bundel Server Components dan menggunakan `require` bawaan Node.js.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ['@acme/ui'],
  },
};

module.exports = nextConfig;
```
