# Membangun Aplikasi Anda

Next.js menyediakan komponen-komponen dasar untuk membuat aplikasi web full-stack yang fleksibel. Panduan-panduan dalam bagian **"Membangun Aplikasi Anda"** menjelaskan cara menggunakan fitur-fitur ini dan cara menyesuaikan perilaku aplikasi Anda.

Bagian-bagian dan halaman-halaman ini disusun secara berurutan, dari dasar hingga tingkat lanjut, sehingga Anda dapat mengikuti langkah-langkahnya secara bertahap saat membangun aplikasi Next.js Anda. Namun, Anda dapat membacanya dalam urutan apa pun atau melompat ke halaman-halaman yang relevan dengan kasus penggunaan Anda.

Jika Anda baru mengenal Next.js, kami sarankan untuk memulai dengan bagian-bagian tentang [Routing][Routing], [Penguraian (Rendering)][Rendering], [Pengambilan Data (Data Fetching)][Data Fetching], dan [Penyusunan Tampilan (Styling)][Styling], karena mereka memperkenalkan konsep-konsep dasar Next.js dan web untuk membantu Anda memulai. Kemudian, Anda dapat menjelajahi lebih dalam ke bagian-bagian lain seperti [Optimisasi][Optimizing] dan [Konfigurasi][Configuring]. Terakhir, ketika Anda sudah siap, periksa bagian-bagian tentang [Penerapan (Deploying)][Deploying] dan Pembaruan [(Upgrading)][Upgrading].
