## API yang Tidak Didukung

Edge Runtime memiliki beberapa pembatasan termasuk:

- API Native Node.js **tidak didukung**. Misalnya, Anda tidak dapat membaca atau menulis ke sistem berkas.
- `node_modules` _dapat_ digunakan, asalkan mereka mengimplementasikan Modul ES dan tidak menggunakan API Native Node.js.
- Memanggil `require` langsung **tidak diperbolehkan**. Gunakan Modul ES sebagai gantinya.

Fitur bahasa JavaScript berikut ini dinonaktifkan, dan **tidak akan berfungsi:**

| API                                                                                                                     | Deskripsi                                                          |
| ----------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| [`eval`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/eval)                               | Mengevaluasi kode JavaScript yang direpresentasikan sebagai string |
| [`new Function(evalString)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Function)       | Membuat fungsi baru dengan kode yang diberikan sebagai argumen     |
| [`WebAssembly.compile`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/compile) | Mengompilasi modul WebAssembly dari sumber buffer                  |

| [`WebAssembly.instantiate`](https://developer.mozilla.org/docs/Web

/JavaScript/Reference/Global_Objects/WebAssembly/instantiate) | Membangun instance modul WebAssembly |
| [`WebAssembly.instantiateStreaming`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/instantiateStreaming) | Membangun instance modul WebAssembly dari streaming byte |

API di atas **tidak tersedia** karena mereka memungkinkan eksekusi kode JavaScript tambahan yang tidak bisa diverifikasi atau terbatas keamanannya.

## Import Dinamis

Anda **dapat** menggunakan `import()` dengan modul ES dalam file JavaScript Anda untuk melakukan impor dinamis. Gunakan `await import()` atau `.then()` untuk mengakses modul yang diimpor.

```javascript
// Misalnya, impor dinamis di Next.js
const dynamicImport = await import("./dynamic-module.js");
```

## API Kustom

Anda dapat mengimplementasikan API kustom Anda sendiri dengan menggantungkan `fetch` dari modul global seperti `window`. Jika Anda ingin menggunakannya di luar Edge Runtime, pastikan untuk mengujinya terlebih dahulu dengan menambahkan kondisi pengecekan global untuk `window` atau `self`. Sebagai contoh:

```javascript
if (typeof window !== "undefined") {
  // Lakukan sesuatu dengan window.fetch di sini
}
```

Anda dapat merujuk pada API API di atas untuk dokumentasi lebih lanjut tentang cara menggunakannya. Semoga berhasil dengan pengembangan Next.js Anda!
