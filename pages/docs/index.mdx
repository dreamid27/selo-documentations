# Introduction

Selamat datang di dokumentasi Next.js!

## Apa itu Next.js?

Next.js adalah kerangka kerja React untuk membangun aplikasi web full-stack. Anda menggunakan Komponen React untuk membangun antarmuka pengguna, dan Next.js untuk fitur-fitur tambahan dan optimalisasi.

Di dalamnya, Next.js juga mengabstraksi dan secara otomatis mengonfigurasi perangkat yang dibutuhkan untuk React, seperti penggabungan, kompilasi, dan lainnya. Ini memungkinkan Anda fokus pada pembuatan aplikasi Anda daripada menghabiskan waktu dengan konfigurasi.

Baik Anda seorang pengembang individu atau bagian dari tim yang lebih besar, Next.js dapat membantu Anda membangun aplikasi React yang interaktif, dinamis, dan cepat.

## App Router vs Pages Router

Next.js memiliki dua router yang berbeda: App Router dan Pages Router. App Router adalah router terbaru yang memungkinkan Anda menggunakan fitur terbaru React, seperti Komponen Server dan Streaming. Pages Router adalah router asli Next.js yang memungkinkan Anda membangun aplikasi React yang dirender di server dan tetap mendukung aplikasi Next.js yang lebih lama.

Di bagian atas panel samping, Anda akan melihat menu dropdown yang memungkinkan Anda beralih antara fitur **App Router** dan **Pages Router**. Karena ada fitur yang unik untuk setiap direktori, penting untuk memantau tab mana yang dipilih.

Breadcrumbs (tanda jalan) di bagian atas halaman juga akan menunjukkan apakah Anda sedang melihat dokumen App Router atau dokumen Pages Router.
