# Next.js Project Structure

Halaman ini memberikan gambaran tentang struktur berkas dan folder tingkat atas dari proyek Next.js. Ini mencakup berkas dan folder tingkat atas, berkas konfigurasi, dan konvensi routing dalam direktori `app` dan `pages`.

## Folder tingkat atas

|                                                                          |                                 |
| ------------------------------------------------------------------------ | ------------------------------- |
| [`app`](/docs/app/building-your-application/routing)                     | Router Aplikasi                 |
| [`pages`](/docs/pages/building-your-application/routing)                 | Router Halaman                  |
| [`public`](/docs/app/building-your-application/optimizing/static-assets) | Aset statis yang akan disajikan |
| [`src`](/docs/app/building-your-application/configuring/src-directory)   | Folder sumber aplikasi opsional |

|                                                                                             |                                           |
| ------------------------------------------------------------------------------------------- | ----------------------------------------- |
| **Next.js**                                                                                 |                                           |
| [`next.config.js`](/docs/app/api-reference/next-config-js)                                  | Berkas konfigurasi untuk Next.js          |
| [`package.json`](/docs/getting-started/installation#manual-installation)                    | Dependensi proyek dan skrip               |
| [`instrumentation.ts`](/docs/app/building-your-application/optimizing/instrumentation)      | Berkas OpenTelemetry dan Instrumen        |
| [`middleware.ts`](/docs/app/building-your-application/routing/middleware)                   | Middleware permintaan Next.js             |
| [`.env`](/docs/app/building-your-application/configuring/environment-variables)             | Variabel lingkungan                       |
| [`.env.local`](/docs/app/building-your-application/configuring/environment-variables)       | Variabel lingkungan lokal                 |
| [`.env.production`](/docs/app/building-your-application/configuring/environment-variables)  | Variabel lingkungan produksi              |
| [`.env.development`](/docs/app/building-your-application/configuring/environment-variables) | Variabel lingkungan pengembangan          |
| [`.eslintrc.json`](/docs/app/building-your-application/configuring/eslint)                  | Berkas konfigurasi untuk ESLint           |
| `.gitignore`                                                                                | Berkas dan folder Git yang diabaikan      |
| `next-env.d.ts`                                                                             | Berkas deklarasi TypeScript untuk Next.js |
| `tsconfig.json`                                                                             | Berkas konfigurasi untuk TypeScript       |
| `jsconfig.json`                                                                             | Berkas konfigurasi untuk JavaScript       |
